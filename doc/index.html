DLDialog 

What is DLDialog?

DLDialog offers the capability to draw a variety of widgets, in order to ease the human user in feeding input to the script. The program is designed to be particularly useful to implement system administration scripts.
You can start DLDialog using different GUI modes, which can be
  • if running in a terminal a turbovision library based GUI
  • if running in a X11 environment a Qt library based GUI
